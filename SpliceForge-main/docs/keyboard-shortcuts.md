# Keyboard Shortcuts Reference

Quick reference for all keyboard shortcuts in SpliceForge.

---

## Canvas Navigation

| Shortcut | Action |
|---|---|
| `Scroll Wheel` | Zoom in / out |
| `Click + Drag` | Pan the canvas |
| `Ctrl + Shift + F` | Fit all nodes to view |
| `Escape` | Clear all traces, cancel bulk connect, deselect all |

---

## Selection & Editing

| Shortcut | Action |
|---|---|
| `Click` | Select a node or edge |
| `Shift + Click` | Add to selection |
| `Drag on empty canvas` | Box-select multiple nodes |
| `Ctrl + A` | Select all nodes |
| `Delete` or `Backspace` | Delete selected nodes or splice |

---

## Copy & Paste

| Shortcut | Action |
|---|---|
| `Ctrl + C` | Copy selected nodes |
| `Ctrl + V` | Paste nodes (offset from original position) |

---

## Undo & Redo

| Shortcut | Action |
|---|---|
| `Ctrl + Z` | Undo last action (DB-synced) |
| `Ctrl + Y` | Redo |
| `Ctrl + Shift + Z` | Redo (alternative) |

---

## Node Positioning

| Shortcut | Action |
|---|---|
| `Arrow Up` | Move selected node 10px up |
| `Arrow Down` | Move selected node 10px down |
| `Arrow Left` | Move selected node 10px left |
| `Arrow Right` | Move selected node 10px right |
| `Shift + Arrow Up` | Move selected node 40px up |
| `Shift + Arrow Down` | Move selected node 40px down |
| `Shift + Arrow Left` | Move selected node 40px left |
| `Shift + Arrow Right` | Move selected node 40px right |

Arrow-key position saves are debounced 300 ms — holding a key for 1 second produces one database write.

---

## Tools & Modes

| Shortcut | Action |
|---|---|
| `Ctrl + B` | Toggle B&W (grayscale) mode |
| `Ctrl + F` | Open / close search |
| `Ctrl + P` | Open command palette (page switcher) |
| `Ctrl + E` | Open export dialog |
| `?` | Open keyboard shortcut reference |

---

## Splicing

| Shortcut | Action |
|---|---|
| `Alt + Shift + C` | Quick bulk splice two selected cable nodes (fiber-by-fiber in order) |

---

## Port Interaction

| Action | How |
|---|---|
| Toggle fiber trace | Click an occupied port |
| Add custom label | Double-click a port, or right-click → Set label… |
| Trace from here | Right-click → Trace from here |
| Clear a single splice | Right-click → Clear splice |
| Continue to another page | Right-click an unoccupied port → Continue to page… |

---

## How to Access the Keymap In-App

Press **`?`** at any time on the canvas, or click the **?** icon in the toolbar to open the interactive keymap overlay.
